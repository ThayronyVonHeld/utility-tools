# 🛠️ UtilityTools (Projeto Canivete)

Aplicação desktop desenvolvida com base em experiências reais na área de operações técnicas, simulando um ambiente de suporte e centralização de ferramentas utilizado em rotinas profissionais.

> ⚠️ Projeto em desenvolvimento contínuo, com implementação incremental de funcionalidades.

---

# 🚀 Tecnologias Utilizadas

### Aplicação Desktop

* Java
* JavaFX

### Ferramentas

* IntelliJ IDEA
* Git / GitHub
* Figma (prototipação de interface)

---

# 🏗️ Arquitetura

O projeto utiliza o padrão **MVC (Model-View-Controller)** para garantir separação de responsabilidades e facilitar a escalabilidade da aplicação.

Além disso, a estrutura é organizada em módulos independentes, permitindo evolução incremental sem acoplamento excessivo.

---

# 💡 Funcionalidades

### Já implementadas

* Timer e StopWatch

### Em desenvolvimento

* Dashboard com atalhos rápidos
* Sistema de notas
* Cofre de senhas
* Central de documentos técnicos
* Lista de contatos com busca
* Ferramentas de tempo (Timer, Stopwatch, Alarm)
* Agenda com calendário
* Integração com sistemas externos
